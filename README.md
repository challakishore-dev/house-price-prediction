# 🏠 House Price Prediction Dashboard

A Machine Learning web application that predicts house prices based on property features such as area, bedrooms, bathrooms, age, parking availability, and location.

Built using **Python, Scikit-learn, and Streamlit**, this project demonstrates how regression models can be used in real estate pricing systems.

---

# 📌 Project Overview

House pricing is one of the most common real-world machine learning problems. Property value depends on multiple factors, and manual valuation is often inconsistent.

This project uses a trained regression model to estimate house prices instantly through an interactive Streamlit dashboard.

Users can enter house details and get a predicted price in real time.

# ▶️ How to Run Locally

## 1. Clone Repository

```bash
git clone  https://github.com/challakishore-dev/house-price-prediction.git 
cd house-price-prediction
```

## 2. Install Dependencies

```bash
pip install -r requirements.txt
```

## 3. Launch App

```bash
streamlit run app.py
```
